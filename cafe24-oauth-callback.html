🚀 다음 단계 — PowerShell에서 시크릿 4개 등록
아래 명령어들을 차례로 실행하고, 각 prompt에 표시된 값을 붙여넣으세요:
firebase functions:secrets:set CAFE24_MALL_ID --project reniv-erp-135a3
→ - 입력
firebase functions:secrets:set CAFE24_CLIENT_ID --project reniv-erp-135a3
→ Client ID 입력
firebase functions:secrets:set CAFE24_CLIENT_SECRET --project reniv-erp-135a3
→ Client Secret 입력
firebase functions:secrets:set CAFE24_REFRESH_TOKEN --project reniv-erp-135a3
→ 위에 표시된 Refresh Token 입력
