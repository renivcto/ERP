📋 3단계 — PowerShell에서 한 번에 실행
아래 명령어 전체를 복사해서 PowerShell에 붙여넣고 Enter.
① Refresh Token 발급 + ② 4개 시크릿 자동 등록 + ③ 함수 자동 재배포까지 한 번에 처리됩니다.

1. 위 "📋 명령어 복사" 클릭
2. PowerShell 창 열고 ERP 폴더로 이동: cd "C:\Users\heeka\OneDrive\0.RENIV\ERP\ERP"
3. 마우스 우클릭으로 붙여넣기 → Enter
4. 명령어가 끝나면 ✅ Refresh Token 발급 + 시크릿 4개 등록 + 함수 재배포 완료
5. Cloud Scheduler에서 fetchCafe24Orders 강제 실행으로 테스트
